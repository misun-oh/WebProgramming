1. 클래스 선언부
public class 클래스명 { }

2. 메서드 선언부
public void 메서드명 (매개변수타입 매개변수명 ) { }

3. 클래스 3가지 요소
전역변수(필드), 생성자, 메서드

4. 생성 키워드
new

5. 반환 없음을 의미하는 예약어
void

6. 반환을 의미하는 예약어
return

7. 메서드의 입력
매개 변수

8. 변수 선언
Type 변수명 = 값;

🍀 리터럴 : 입력되는 값 그 자체를 리터럴이라고 부릅니다!

9. Type의 종류
int num = 10; // 정수
double num2 = 3.14; // 실수
boolean isTrue = false; // 논리값(true, false)
String str = "안녕하세요"; // 문자열

🍀 Type은 기본타입과 참조타입으로 나눠지며 기본타입을 제외한 나머지는 모두 참조타입 이다.

10. 메서드 안에 메서드를 작성할수 없다.

11. 문장의 끝은 ; 으로 한다

12. Outline 뷰를 확인하면 메서드의 정보 알수 있다
🍀 메서드를 사용하기 위해서는 메서드의 이름과 매개변수의 타입, 갯수를 알아야 한다.
